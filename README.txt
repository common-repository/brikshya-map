== Frequently Asked Questions ==

= What Should I do first after installing the plugin? =

After installing the Plugin user have to save the google map api key, until the key is saved the map will not properly, may cause errors.

= How to add Marker? =

User can add their won marker icon. That will be loaded while loading map on website. Multiple Markers can be added, however user can currently add markers or  edit the name of the marker. The current version  doesn't support editing the marker icon or image once it is added.

= How to add Map? =

From Briskshya Map >  Add New, users can add new map, the user can drag the first marker shown  to make the default map address which will be shown when the map is loaded.

= Can Multiple map be added on single page? =

No, sorry the current version doesn't support multiple map on a single page.

= How to add map on page? =

Users can add the map with the help of sortcode that will be displayed on the Brikshya Map Main Page
